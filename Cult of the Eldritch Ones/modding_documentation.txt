----
Triggers:
--
is_any_bad_priest
  - checks if any type of wicked priest
eldritch_ai_initiative_restricted
  - shortcut to check if gamerule AI Initiative is set to restricted
eldritch_ai_initiative_increased
  - shortcut to check if gamerule AI Initiative is set to increased

----
Effects:
--
generate_rng_10.1
generate_rng_10.2
  - creates a random number between 1-9 and saves it as value of @ROOT.rng_value_10.1 (.2)
generate_rng_100
  - creates a random number between 1-100 and saves it as value of @ROOT.rng_value_100
  tip: can be used to insert a number into localisation by using [@ROOT.rng_value_100.GetValue]

[ROOT.GetDemocraticMisconduct]
  - scripted_localisation: fills in a random (democratic-ish) misconduct that a government official could have committed
